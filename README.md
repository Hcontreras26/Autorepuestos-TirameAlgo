## 📁 Estructura del Proyecto

*   **/admin**: Panel de administración para el control total del sistema.
*   **/panelCliente**: Interfaz dedicada para la gestión de usuarios/clientes.
*   **/logica**: Scripts de backend que manejan las operaciones del negocio.
*   **/reportes**: Módulo para la generación de informes detallados.
*   **/login-sesion**: Gestión de autenticación y seguridad.

## 🛠️ Instalación

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/Hcontreras26/Autorepuestos-TirameAlgo.git](https://github.com/Hcontreras26/Autorepuestos-TirameAlgo.git)
    ```
2.  Instala las dependencias de Composer:
    ```bash
    composer install
    ```
3.  Importa el archivo de base de datos `repuestos_tiramealgo.sql` en tu servidor MySQL local.
4.  Configura tus credenciales de conexión a la base de datos en los archivos correspondientes dentro de `/logica`.

## 📈 Funcionalidades Principales

- [ ] Control de inventario de repuestos.
- [ ] Gestión de usuarios con roles diferenciados (Admin/Cliente).
- [ ] Generación automática de reportes.
- [ ] Sistema de login seguro.
